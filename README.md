# paymentGatewayIntegration
Integration of Payment Gateway


To run this project : 
First, clone this repository, then run the following commands.

To run backend : 
cd .\backend\
npm install

create a .env file in backend folder and all these values(although it's not recommended to publicly display the values of this file) :
PORT=5000
MERCHANT_KEY_CREATE=$2y$10$VqOdRRE21p6/mGJLjyglk.J7zFM7tusWPQRcVtG7fbCjV9lsSTREO


Then run this command : nodemon server.js
Your backend will be up and running


To run frontend : 
cd .\frontend\
cd .\eCart\

npm install

and then run npm run dev

your frontend will be up and running on localhost://5173
